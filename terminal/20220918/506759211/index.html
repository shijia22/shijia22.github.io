使用 Oh-My-Posh 更改 Bash treminal

發表於 2022-09-18 更新於 2022-09-19 分類於 Terminal

契機就是看到 MacOS 系統都有美美的 treminal，但是 Windows 能夠適用的稍微少了些，所以就繼續一下改裝自己 Bash Terminal 的過程。

前言

先把字型裝起來。
我本身不管是 Windows or MacOS 都是使用 Bash，因此下方步驟則會以此為主。

MacOS

記得先安裝 Homebrew

非 M1M2

  1. 使用指令安裝 OhMyPosh
  2. 下載自己喜歡的 Theme 存在自己知道的地方，像我就是存在 finder/Documents
  3. 原本 MacOS 預設都是 ZSH，不過我有改成 Bash 因此指令就是

M1M2

M1M2 操作上有點特別，如果事前沒有處理過，直接進行安裝可以參考以下步驟：

  1. 依據安裝 rosetta 的 M1M2 步驟
  2. 使用 arch -x86_64 安裝 oh-my-posh

參考資料：Error: Cannot install in Homebrew on ARM processor in Intel default prefix (/usr/local)

Windows

  1. 使用 Winget 來進行安裝，開啟 Git Bash 時記得使用「系統管理員」開啟
  2. 途中會詢問一些內容，type Y
  3. 因為我使用的是 Git Bash 因此指令就是

參考資料：Oh My Posh、Oh My Posh：全平台终端提示符个性化工具 、我選用的主題
