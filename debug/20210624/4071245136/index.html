解決 Windows 安裝 NVM 失敗問題

發表於 2021-06-24 更新於 2022-09-06 分類於 Debug

第一次進行 Windows 環境設定與安裝時，沒有想過會失敗的問題。

前言

剛開始寫程式時，不免俗的會安裝許多不同的程式等，我就是那個比較常會需要磨練（Debug）新手運的那種人ಥ_ಥ

問題

錯誤訊息：could not be found or does not exist. Exiting.

could not be found or does not exist. Exiting.

重現步驟

  1. 至 nvm-windows 的 github 下載

    下載並解壓縮

  2. 安裝應用程式

記得這裡要「同意」

這裡預設會是 C 槽使用者的資料夾

這張圖 ↑ 就是失敗的主因，我使用的電腦是使用 Microsoft 帳戶直接登入，而我預設的「使用者名稱」是中文，當時我想，安裝應用程式使用程式的預設通常不會有問題，我真是太天真惹 (☍﹏⁰)

安裝 Tips：

  1. 資料夾名稱不能有空白。 e.g. C:\Users\shi jia
  2. 資料夾名稱不能有中文（因為會有編碼問題）。e.g. C:\Users\釋迦
  1. 開啟 Git Bash 輸入 nvm version，確認畫面
