Git Bash 創建 Vue 可以上下移動的神奇關鍵

發表於 2022-02-18 更新於 2022-09-06 分類於 Git

Bash 好用歸好用，但一直有個問題上下移動感受不出來到底有沒有在移動，快來看看這次我發現了什麼酷東西！

前言

之前剛學 Vue 的時，建立專案都會透過 Git Bash 操作，但 N 次後發現真的非常奇怪，所以後來都改用 VS code 直接操作 XD。

但就在剛剛，突然發現一個神奇的指令 winpty vue.cmd create hello-world 竟可以神奇的上下移動了！

後來發現僅有在 Git bash 建立 Vue 專案時 vue create hello-word 改成 winpty vue.cmd create hello-world 才可以成功上下移動，而本質上改的是 Vue 在指令列操作的狀況，不過可以上下移動真的是一大進步。
